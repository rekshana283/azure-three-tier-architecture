# Azure Three-Tier Architecture Design

## Project Overview
This project demonstrates a basic Microsoft Azure three-tier architecture for hosting a web application. It explains how Azure services work together to build a secure, scalable, and reliable cloud solution.

## Architecture Components

- Azure Virtual Network (VNet)
- Network Security Group (NSG)
- Azure Load Balancer
- Azure Virtual Machine (VM)
- Azure SQL Database
- Azure Storage Account

## Architecture Flow

User
↓
Internet
↓
Azure Load Balancer
↓
Azure Virtual Machine
↓
Azure SQL Database
↓
Azure Storage Account
